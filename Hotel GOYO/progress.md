### 페이지 구조 (Option A + B 통합)

**Option A — 원페이지 스크롤 구조**
- 사이트 헤더: 이름, 직함, 이메일
- 프로젝트 인덱스: 001 Hotel GOYO / 002 — / 003 — (앵커 링크)
- 프로젝트 섹션: 좌측 정보 패널 + 우측 이미지 그리드, 스크롤로 연속 이어짐
- 002, 003은 다음 프로젝트 자료 추가 대기 중 (플레이스홀더)

**Option B — 이미지 클릭 슬라이드쇼 (Lightbox)**
- 이미지 클릭 시 풀스크린 라이트박스 열림
- ← → 화살표 또는 키보드로 이전/다음 탐색
- ESC 또는 Close 버튼으로 닫기
- 현재 이미지 번호 / 전체 카운터 표시
